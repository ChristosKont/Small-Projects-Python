Στη σειρά ασκήσεων αυτή θα συμπληρώσετε τον κώδικα Python που βρίσκεται στο αρχείο hw1.py, ως εξής:

  1. Κατεβάστε το αρχείο hw1.py στον υπολογιστή σας. (Θυμηθείτε τον κατάλογο όπου τοποθετήθηκε.)
  2. Εκκινήστε κάποιον text editor (πχ, Notepad++) και φορτώστε το hw1.py (File, Open)
  3. Το αρχείο που βλέπετε περιέχει συναρτήσεις των οποίων το σώμα πρέπει να συμπληρώσετε.
  4. Συμπληρώστε τα κενά (_____) ή κώδικα σύμφωνα με τις περιγραφές που σας δίνονται σε σχόλια (ανάμεσα στα """ .... """"). Στα σχόλια κάθε συνάρτησης δίνονται παραδείγματα για την αναμενόμενη τιμή ή παρενέργεια (side-effect) για διάφορες τιμές ορισμάτων.
  5. Όταν τελειώσετε με τις προσθήκες σας, αποθηκεύστε τις αλλαγές (στο ίδιο αρχείο hw1.py). (Δεν χρειάζεται να συμπληρώσετε το σώμα όλων των συναρτήσεων πριν προχωρήσετε σε έλεγχο ορθότητας/δοκιμές, μπορείτε να το κάνετε για μια συνάρτηση τη φορά.)

Μπορείτε να καλέσετε τις συναρτήσεις στο hw1.py από το διαλογικό περιβάλλον της Python ως εξής:

  1. Εκκινήστε το διαλογικό περιβάλλον (εντολή python ή python3 στη γραμμή εντολών) από τον ίδιο κατάλογο που περιέχεται το hw1.py
  2. Για να μπορείτε να καλέσετε όλες τις συναρτήσεις γράψτε (μετά τα >>>): >>> from hw1 import *. Εάν θέλετε να δοκιμάσετε μόνο μια συνάρτηση, πχ., τη sum_powers, γράφετε: >>> from hw1 import sum_powers
  3. Δοκιμάστε αν οι συναρτήσεις έχουν σωστή συμπεριφορά, πχ, δίνοντας >>> sum_powers(2,2 3,2) θα πρέπει να εμφανιστεί η τιμή 13

Αν εμφανιστεί μήνυμα λάθους από τον διερμηνευτή ή η συνάρτηση δεν κάνει τον σωστό υπολογισμό/παρενέργεια, επιστρέψτε στον text editor και αλλάξτε τον κώδικά σας. Έπειτα, πάλι αποθηκεύστε τις αλλαγές και δοκιμάστε τον κώδικα στο διαλογικό περιβάλλον όπως παραπάνω, κοκ. Συνεχίστε μέχρι οι συναρτήσεις να έχουν τη σωστή συμπεριφορά.

Παραδίδετε τις απαντήσεις σας ανεβάζοντας στο eclass (μέσα από τον χώρο εργασιών) μόνο το τελικό hw1.py αρχείο σε συμπιεσμένη μορφή zip ή rar ή μέσα σε συμπιεσμένο φάκελο (βλ. σχετική ανακοίνωση στις 'Ανακοινώσεις'). (Άλλα αρχεία ή συνοδευτικά σχόλια αφαιρούνται αυτόματα.)

Μπορείτε να παραδίνετε απαντήσεις (με τον παραπάνω τρόπο) χωρίς να έχετε συμπληρώσει το σώμα όλων των συναρτήσεων.

Πριν παραδώσετε τις απαντήσεις, βεβαιωθείτε ότι η εντολή from hw1 import * του διερμηνευτή δεν βγάζει μήνυμα λάθους (όταν εκτελεστεί σωστά, δηλ., το hw1.py βρίσκεται στον τρέχοντα κατάλογο.). Σε διαφορετική περίπτωση, ο βαθμός σας στη σειρά ασκήσεων αυτή θα είναι 0.

Σημείωση 1: ο αριθμός των χαρακτήρων (_) στα κενά δεν υποδηλώνει κάτι. Οι απαντήσεις σας μπορούν να χρησιμοποιούν περισσότερα ή λιγότερο χώρο. Αφού συμπληρώσετε την απάντησή σας, σβήστε τυχόν _ αλλιώς θα λάβετε μήνυμα λάθους από τον διερμηνευτή καθώς αποτιμάτε τις συναρτήσεις.

Σημείωση 2: στο σώμα της συνάρτησης my_abs, δεν επιτρέπεται να χρησιμοποιήσετε την ενσωματωμένη συνάρτηση abs. Μπορείτε να χρησιμοποιήσετε άλλες ενσωματωμένες συναρτήσεις, καθώς και συναρτήσεις από το module operators και math, πχ, max, min, pow, add, sub, mul, truediv, floordiv, sqrt,...
